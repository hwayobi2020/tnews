# 사용자 가이드: 나만의 뉴스봇 만들기

## 🎯 서비스 개요

**당신이 제공하는 것:**
- ✅ 뉴스 수집 및 AI 요약 서비스 (무료 또는 유료)
- ✅ 중앙 서버에서 모든 처리

**사용자가 하는 것:**
- ✅ 자기만의 텔레그램 봇 만들기 (2분)
- ✅ 봇 토큰 등록 및 키워드 설정

---

## 📱 Step 1: 텔레그램 봇 만들기 (2분)

### 1.1 BotFather 찾기

1. 텔레그램 앱 열기
2. 검색: `@BotFather`
3. 파란색 체크마크 확인 후 선택

### 1.2 봇 생성

**BotFather에게 보내기:**
```
/newbot
```

**봇 이름 입력 (자유):**
```
내 AI 뉴스
```

**봇 username 입력:**
```
my_ai_news_bot
```
> 규칙:
> - `bot` 또는 `Bot`로 끝나야 함
> - 영문, 숫자, 언더바(_)만 사용
> - 이미 사용 중이면 다른 이름 시도

### 1.3 토큰 복사

성공하면 이런 메시지가 옵니다:

```
Done! Congratulations on your new bot.
...
Use this token to access the HTTP API:
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

**🔐 이 토큰을 복사하세요!**

## 🌐 Step 2: 서비스 등록

### 2.1 등록 웹페이지 접속

```
https://your-service.com/register
```
> (실제 URL은 관리자가 제공)

### 2.2 정보 입력

**등록 폼:**
```
📱 봇 토큰: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
📝 키워드: 인공지능
⏰ 알림 시간: 09:00
```

**제출 버튼 클릭!**

---

## ✅ Step 3: 봇 활성화

### 3.1 봇에게 메시지 보내기

1. 텔레그램에서 방금 만든 봇 검색
2. `/start` 메시지 전송

### 3.2 확인

웹페이지에서 "활성화 완료!" 메시지 확인

---

## 📰 Step 4: 뉴스 받기

**다음날 오전 9시:**
- 봇으로부터 뉴스 브리핑 도착!
- AI가 요약한 키워드 뉴스 10개

**예시:**
```

📰 핵심 뉴스 TOP 5

2. [제목] - 한 줄 요약
...
```

## ❓ FAQ

### Q: 봇을 꼭 만들어야 하나요?

**A:** 네! 개인정보 보호를 위해 각자의 봇으로 받습니다.
- 공용 봇: 다른 사람의 뉴스도 볼 수 있음
- 개인 봇: 본인만 접근 가능

### Q: 봇 만들기가 어려워요

**A:** 2분이면 됩니다! 위 가이드 따라하세요.
- BotFather 찾기 → /newbot → 이름 입력 → 끝!

### Q: 토큰을 잃어버렸어요

**A:** BotFather에게 다시 받을 수 있습니다:
```
/token (봇 선택)
```

### Q: 뉴스가 안 와요

**A:** 체크리스트:
1. ☑️ 봇에게 `/start` 메시지 보냈나요?
2. ☑️ 웹페이지에서 "활성화 완료" 확인했나요?
3. ☑️ 알림 시간이 지났나요?
4. ☑️ 봇을 차단하지 않았나요?

### Q: 키워드를 여러 개 설정하고 싶어요

**A:** 프리미엄 플랜 출시 예정입니다!
